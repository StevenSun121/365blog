win10不为人知的快捷键

2018-03-14
优化
windows
  1. Shift + win + S 系统级截屏 , 其他截屏软件截不到图的可以试试这个右键重新截屏
  2. Ctrl + win + C 反转屏幕颜色 , 再按一次恢复
  3. win + U 轻松使用
  4. Ctrl + win + N 打开讲述人设置页面
  5. Ctrl + win + M 打开放大镜设置页面
  6. win + + / - 放大镜倍率
  7. win + ESC 退出放大镜
  8. win + W 打开工作区 , 里面的小工具还是不错的
  9. win + G 游戏屏幕录制
  10. Alt + Enter 当前应用全屏 , 小伙伴们用 F11 不能全屏的可以试试这个
  11. win + X 快速打开快捷菜单
  12. Ctrl + win + D 创建虚拟桌面
  13. Ctrl + win + F4 关闭当前虚拟桌面
  14. Ctrl + win + ← / → 切换虚拟桌面
  15. win + , 临时查看桌面
  16. win + M 最小化所有窗口
  17. Shift + win + M 还原上一个快捷键最小化的窗口
  18. Alt + ESC 以应用打开的顺序切换
  19. Alt + Space 打开窗口菜单
  20. Shift + F10 右键菜单
  21. win + A 操作中心
  22. win + i 设置
  23. win + T 切换任务栏应用
  24. win + K 打开连接
  25. win + Q / S 打开Cortana
  26. LOL选英雄时连按两次 alt + F4 可以看对面选的英雄和天赋符文
  27. 新建一个记事本，用Notepad输入联通→保存→关闭再打开
